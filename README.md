# Minimal conchat_apk

Minimalistic Android application conchat lepra chat

## Requirements

- Android Studio Iguana or newer
- Android SDK 34
- JDK 17

## Run

1. Open `conchat_apk` in Android Studio.
2. Wait for Gradle sync.
3. Select an emulator or device.
4. Click Run.

## Project Structure

- `app/src/main/java/com/example/minimalapp/MainActivity.kt` - app entry point and UI.
- `app/src/main/java/com/example/minimalapp/ui/theme/` - Compose theme files.
- `app/src/main/AndroidManifest.xml` - Android manifest.
